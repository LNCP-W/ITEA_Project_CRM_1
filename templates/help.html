Создание департамента:

Откыть страницу "Все Департаменты" ==> нажать кнопку "создать департамент" ==> ввести название, локацию и телефон ==> нажать кнопку "создать"


Регистрация сотрудника:

Откыть страницу "Департаменты" ==> нажать на id департамента в каком хотите создать сотрудника => нажать кнопку "Регистрация Сотрудника" ==> ввести имя, должность и телефон ==> нажать кнопку "создать"


Регистрация клиента:

Откыть страницу "все клиенты" ==> нажать кнопку "Регистрация Клиента" ==> ввести имя и телефон ==> нажать кнопку "создать"


Регистрация Заявки:

Откыть страницу "Все сотрудники" ==> нажать на id сотрудника на которого хотите зарегистрировать заявку => нажать кнопку "Регистрация заявки" ==> выберите клиента который обращается (если клиента нет в списке смотрите пункт Регистрация клиента), виберите статус и тип заявки а также введите описание, Серийный Номер и если извесно - цену ремонта ==> нажать кнопку "создать"


Редактирование департамента, сотрудника, клиента, заявки:

Откыть страницу с нужным содержанием ==> нажать на id записи которую хотите отредактировать=> внеситье необходимые изминения ==> нажать кнопку "изменить"
!!!ВНИМАНИЕ!!! некоторые поля невозможно изменить


Удаление департамента, сотрудника, клиента, заявки:

Откыть страницу с нужным содержанием ==> нажать на id записи которую хотите отредактировать => нажать кнопку "удалить" => если уверенны в своих действиях - нажать "ОК" во всплывающем окне
!!!ВНИМАНИЕ!!! при удалении департамента, сотрудника, клиента будут удалены все привязвные к ним записи
